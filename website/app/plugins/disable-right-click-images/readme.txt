Disable right click on images in WordPress

== Description ==

This plugin disables right click on images in WordPress. All you need to do is activate the plugin and visitors on your site will not be able to right-click on any image.

https://www.youtube.com/watch?v=L2rHC2AKq-4&rel=0

This is very useful for websites where you have a lot of images and you don't want visitors to easily save them by right-clicking on it.

For detailed documentation please visit the [Disable Right Click Images](https://noorsplugin.com/disable-right-click-images-wordpress/) plugin page

== Installation ==

1. Go to the Add New plugins screen in your WordPress Dashboard
1. Click the upload tab
1. Browse for the plugin file (disable-right-click-images.zip) on your computer
1. Click "Install Now" and then hit the activate button
